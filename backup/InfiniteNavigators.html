The About:Blank Page


The About:Blank Page Is The Page That Allows You To Open This Website Or Any Others
In About:Blank

------Buttons------

When You Click Open This Site, It Will Open This Site In About:blank

When You Click Open Custom Url, It Will Verify The Url And Do One Of Three Things
If The Url Allows CORS And Is A Valid Website, It Will Automaticlly Open In About:Blank
If The Url Does Not Allow CORS And Is A Valid Website, A Warning Will Appear That Says
This Website Does Not Allow CORS, So It May Not Work.
If The Website Entered Does Not Exist, It Will Say That The Website Does Not Exist And
Will Not Allow You To Open It In About:Blank
