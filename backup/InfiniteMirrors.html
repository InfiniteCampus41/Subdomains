Mirror Sites



This Page Will Tell You How To Set Up A Mirror Link Of Infinite Campus



Mirror Sites Rules:
By Creating A Mirror Site Of Infinite Campus, You Agree To The Following:

  • You Will NOT Attempt To Bypass The Current Database Rules
  • You Will NOT Steal User's Data
  • You Will NOT Attempt To Abuse The Current API
  • You Will NOT Claim To Be The Owner Of The Site
  • You Will NOT Redirect Users To Your Own Payment Method In The Donation Page


To Create A Mirror Link Of This Site, Go To The Github Repo And Fork It Github Repo


Or, Go to The Download This Website Page Under The Download This Website Tab


Note: This Next Section Is REQUIRED If You Want Custo PFPs To Work In The Chat
Step 1: Delete The pfps Folder
Step 2: Find Anywhere In The Files account.js, adminchat.js, And chat.js Where A pfp Link Is Referenced
(Just Search For /pfps In These Files)
Step 3: Before Each /pfps, Insert This Link: https://raw.githubusercontent.com/InfiniteCampus41/InfiniteCampus/refs/heads/main
The pfps Link Should Now Look Something Like: https://raw.githubusercontent.com/InfiniteCampus41/InfiniteCampus/refs/heads/main/pfps/index.json?t=${Date.now()}
Now, If You Open The Site, If raw.githubusercontent.com Links Are Not Blocked, Custom Pfps Should Show Up


Another Important Thing.
The Proxy Will NOT Work On Your Mirror, Unless You Are Fully Deploying The Website With The Full Server Code.
This Is A Very Complicated Process And I Can't Even Easily Say How To Deploy It.
However, Some Perks Of Fully Deploying The Site Are:
Working And Local PFPs
Auto Updating Git Repo
Working Proxy
Working Games Method 1 (Will Have To Modify Code To Get Button To Show Up)
Able To Block URLs For The Proxy With Your Own Admin Password

Bring Your Own Domain



BYOD, Or Bring Your Own Domain Is A Way To Have Your Own domain And Have It Point To Infinite Campus



How To Set Up

Step 1: Create A CNAME Record For The Subdomain, Or Domain That You Want To Be Infinite Campus
Step 2: Have The CNAME Go To infinitecampus.xyz
That's It!
