Roles Info


Infinite Campus Has Roles. These Roles Are Only Appliccable If You Are Logged In.


To Log In, Click Here

These Roles Are Ranked From Most Powerful To Least

Permissions From Lower Power Roles Also Carry To Higher Power Roles

Ex: If Premium T1 Can Have 500MB File Uploads, Every Role Above That Also Gets It.

The Only Permissions That Do Not Carry Are:

  • Custom Subdomain - Premium T3
  • Manage Website Server - Developer




Role #1: Owner

This Role Is For The Owner Of The Site
Permissions:

  • Same As Tester


Role #2 Tester

This Role Is For Test Accounts
Permissions:

  • Read And Write To Everything


Role #3 Co-Owner

This Role Is For People That Are Very Trusted By The Owner To Moderate
Permissions:

  • Create Channels
  • Delete Channels
  • Rename Channels
  • Verify Users
  • Manage All Typing Data
  • Manage Server
  • Accept Movie Applicants


Role #4 Head Admin

This Role Is For People That Are Trusted To Moderate
Permissions:

  • Same As Admin


Role #5 Admin

This Role Is For People That Are Trusted By The Owner, But Are Not Active On The Site
Permissions:

  • Delete Messages Sent By Roles Below
  • Mute Roles Below


Role #6 Developer

This Role Is Given To People That Are Coding Infinite Campus
Permissions:

  • Read User's Emails
  • Add Website Updates
  • Add Future Updates
  • Manage Website Server


Role #7 Premium T3

This Role Is Given To Users After Donating $15 Or More In One Donation To The Site
Permissions:

  • Access Admin Chat
  • Make Your Own Subdomain Of Infinitecampus.xyz

    • Example: name.infinitecampus.xyz
    To Get Your Subdomain, You Must Contact The Owner


Role #8 Premium T2

This Role Is Given To Users After Donating $10-$14.99 In One Donation To The Website
Permissions:

  • Access Premium Chat


Role #9 Premium T1

This Role Is Given To Users After Donating $5-$9.99 In One Donation To The Website
Permissions:

  • 500MB File Uploads In 5 Min File Upload App
  • Files Last 15Mins Instead Of 5 In 5 Min Upload App


Role #10 100th User

This Role Will Be Given To The 100th User That Signs Up For The Website
Permissions:

  • Same As Regular User


Role #11 Partnered User

This Role is Given To Users That Partner With Infinite Campus
Permissions:

  • Same As Regular User


Role #12 Verified User

This Role Is Given To You Once Someone Verifies You
Permissions:

  • See Unrestricted Channels
  • Send Messages In Unrestricted Channels
  • Delete Own Messages
  • Edit Own Messages
  • Create Private Chats
  • Mention Users
  • Change Own Typing Data
  • Read Who Is Typing


Role #13 Unverified User

When You First Sign Up For The Website, You Will Be An Unverified User.
To Verify, You Just Need To Wait. It Usually Takes At Most 12H To Verify
Permissions:

  • Read Own Private Chats
  • Read Own Mentions
  • Change/Set Own Pic
  • Change/Set Own Display Name
  • Change/Set Own Name Color
  • Change/Set Own Password
  • Change/Set Own Bio


Role #14 Spam User

This Role Is Given To People That Have Engaged In Suspicious Activity On The Website
If You See Someone With This Role, Please Do Not Interact With Them
Permissions:

  • Same As Unverified User Unless Verified


Role #15 Signed Out User

This Role Is What You Have If You Are Viewing The Site Not Signed In
Permissions:

  • Read User Bios
  • Read User Pics
  • Read User Name Color
  • Read User Display Names
  • Read User Roles
  • Read Website Updates
  • Read Website Future Updates


Role #16 Discord User

This Role Is Given To You If You Enter In Your Discord Username In The Settings Page
Permissions:

  • Same As Regular User


Role #17 Donater

This Role Is Given To You If You Donate Less Than $5 To The Website
Permissions:

  • Same As Regular User


Role #17 Guesser

This Role Is Given To You If You Complete A Seceret Task Somewhere Involving The Site.
This Role Is No Longer Available
Permissions:

  • Same As Regular User
