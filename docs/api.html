POST /upload

Uploads A File Under 10MB To The Discord
Example File Upload Code 

                    
    uploadForm.addEventListener('submit', e=>{
        e.preventDefault();
        if(!fileInput.files.length) return showError('No File Selected');
        const formData = new FormData();
        formData.append('file', fileInput.files[0]);
        formData.append('channelId', currentChannelId);
        fetch(`${backendUrl}/upload`, {
            method:'POST',
            body:formData
        }).then(res=>{
            if(!res.ok) res.text().then(text=>showError('Upload Failed:'+text));
            fileInput.value=''; fileLabel.textContent='Select A File';
        })
    });

GET /weather


To Get The Weather Of A Location, You Must First Get The User's City And State
You Then Need To Send It In This Url Format
https://api.infinitecampus.xyz/weather?city=${city}&state=${state}
Example:
https://api.infinitecampus.xyz/weather?city=New%20York&state=New%20York


Example JSON Response: 

                        
    {
        "location": "New York, New York",
        "temperature": {
            "fahrenheit": 39,
            "celsius": 4
        },
        "finalDigit": {
            "digit": 0
        },
        "condition": "Cloudy",
        "emoji": "☁️",
        "display": "New York, New York:☁️ 39",
        "station": "KNYC",
        "time": "2026-02-11T20:51:00+00:00"
    }

GET /movies/x9a7b2/:name

This Endpoint Is For Streaming Movies

It Just Returns The File

GET /list_videos_x9a7b2

This Endpoint Is For Listing The Available Movies
It Returns The Full File Name As "file"
It Returns The File Name Without .mp4 at "name"
It Returns The Size In Bytes At "size"
It Returns The Time The Video Was Uploaded At "mtime"
It Returns The Formatted Size In Kb, Mb, Gb, Or TB At "humanSize"
It Returns The Order Of The Movie In The List At "order"
Here Is An Example Response 

                    
    {
        "ok": true,
        "videos": [
            {
                "file": "Fnaf.mp4",
                "name": "Fnaf",
                "size": 254112313,
                "mtime": "2026-02-06T09:25:19.899Z",
                "humanSize": "242.3 MB",
                "order": 10
            },
            {
                "file": "Fnaf2.mp4",
                "name": "Fnaf2",
                "size": 254548186,
                "mtime": "2026-02-06T20:57:41.093Z",
                "humanSize": "242.8 MB",
                "order": 20
